# Precision Oncology for Breast Cancer Diagnosis
Master’s Thesis Research Repository

## Creation of Metabolic Models

This folder contains all the code used to process transcriptomic data files and convert them to TXT format. It also includes MATLAB scripts with all the parameters for constructing thermodynamic genome-scale metabolic models. Additionally, it contains analysis scripts in Python, MATLAB, and R, as well as the corresponding visualizations and graphs.
